# CV Assets

This directory is intentionally minimal in the public template.

## What is included

- `CV.tex` is a public-safe placeholder LaTeX CV skeleton.
- No private resume content or rendered personal PDF is included here.
- Add your own `CV.pdf` only if you want to expose a downloadable CV on the site.

## Recommended workflow

1. Edit `CV/CV.tex`.
2. Export `CV/CV.pdf`.
3. Set `author.cv` in `_config.yml` to the public file path.
4. Rebuild the site and verify the CV link from the homepage.

## Attribution

The fuller academic CV style used in the private source site traces back to Jake Gutierrez's "Jake's Resume" LaTeX resume template.

If you reuse that lineage or redistribute a derived CV template, keep the relevant attribution and license notice.
